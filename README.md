Este repositorio integra las ejercitaciones realizadas en las Clases 11, 12, 13 y 14, del curso de Frontend con Ia. 
Corresponden al Módulo 4. 
El Index núclea los enlaces de las 4 clases e intenta mostrar las ejercitaciones correspondientes a cada uno.Agregar documentacion para saber de que se trata este repositorio y como se usa , etc

#Clase 11
Trabajamos en el desarrollo de un cronómetro en pantalla. 
El mismo tiene 3 botones, INICIO - PAUSAR - REINICIAR.
La interfaz es oscura y cuenta con una imagen para la pestaña (favicon).
Cuenta con una función que permite que el reloj cambie de color a los 5 segundos y emita un sonido.

#Clase 12
Trabajamos en el desarrollo de una pagina de clima, con el uso de una api publica. 
En la misma podemos cargar la ciudad y nos trae el clima para esa ciudad junto con una imagen asociada al tipo de clima.

#Clase 13
Trabajamos en desarrrollo de una pagina para buscador de con el uso de una api publica de git hub. 
En la misma podemos buscar usuarios.
